# Objetivo del usuario

Escribe aquí lo que quieres que consiga el repo.

## Cómo escribirlo

Hazlo de forma simple y clara. Puedes escribir:

- un único objetivo
- varios objetivos relacionados
- restricciones
- prioridades
- cambios respecto a instrucciones anteriores

## Reglas para ti como usuario

- Puedes editar este archivo en cualquier momento.
- La instrucción más reciente tiene prioridad sobre una antigua si se contradicen.
- No necesitas usar lenguaje técnico.
- Si quieres cambiar el rumbo, dilo claramente aquí.

## Ejemplo

Quiero que el repo construya una solución gratuita y sin billing para [...].
Prioridad máxima: [...].
No quiero que use [...].
Si necesitas pedirme algo, explícamelo paso a paso.
